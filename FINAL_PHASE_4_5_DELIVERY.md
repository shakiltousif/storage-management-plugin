## 🚀 GETTING STARTED

### 1. Enable Portal
Add to any page:
```
[royal_storage_portal]
```

### 2. Enable Checkout
Add to checkout page:
```
[royal_storage_checkout]
```

### 3. Configure Payment
1. Install WooCommerce
2. Configure payment gateway
3. Set payment methods

### 4. Test
1. Log in as customer
2. Visit portal
3. Test all features
4. Test payment processing
